# Code of Conduct

## Our Pledge

This is a small personal / project-focused repository. Interactions are welcome from anyone who engages in good faith.

## Standards

**Expected:**

- Be respectful and direct. Disagreement is fine; personal attacks are not.
- Keep discussion on-topic. Technical questions, bug reports, suggestions, and pull requests are all welcome.
- Assume good intent. Ask for clarification before escalating.

**Not acceptable:**

- Harassment, insults, or personal attacks.
- Publishing others' private information (doxing).
- Spam, promotion unrelated to the project, or off-topic content.
- Sustained disruption of discussion.

## Enforcement

The maintainer may edit, remove, or reject any comment, commit, issue, or pull request that does not align with this Code of Conduct, and may ban participants from further interaction. No formal appeals process is defined — this is a personal repository, not a managed community.

## Scope

This Code applies within project spaces (repository issues, pull requests, commits, and directly-linked discussions) and to reports of behaviour elsewhere that directly affects these spaces.

## Reporting

Concerns can be reported privately via GitHub's Report abuse feature on the specific comment, or by opening a private Security Advisory if public reporting would amplify harm.
